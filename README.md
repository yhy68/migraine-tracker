# 偏头痛记录工具

偏头痛发作记录工具，追踪每一次发作的诱因、症状和用药效果。

## 使用方式

直接访问 GitHub Pages 地址，无需任何配置，打开即用。数据自动同步到私有仓库。

## 数据隐私

所有记录保存在：
- 浏览器本地（LocalStorage）
- 私有 GitHub 仓库 `yhy68/migraine-data`（自动同步）

## 功能

- 记录发作：疼痛程度、部位、类型、前驱症状、伴随症状、诱因、用药
- 历史列表：搜索、筛选、编辑、删除
- 数据同步：自动同步到 GitHub 私有仓库
- 导入导出：JSON 文件备份，跨设备迁移

## 技术

纯静态 HTML/CSS/JS，GitHub Pages 部署，GitHub API 数据同步。
